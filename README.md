# fpga-practice
This is a fpga project on DE10-Lite board.

# Setup
## Install Quartus Prime Lite

## Install USB Blaster driver
C:\intelFPGA_lite\18.1\quartus\drivers\usb-blaster\usbblstr.inf


## Device information
MAX 10 10M50DAF484C7G
